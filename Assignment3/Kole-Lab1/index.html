Console Commands

 JsonParser.fromJson(tournamentJsonString) 
 Accepts json string representing golf tournament, deserializes to javascript objects. 
 XmlParser.startParsing(xml) 
 Accepts xml dom object, url to a xml document, or string representing a golf tournament, deserializes to javascript objects. 
 validateData() 
 Returns true or false. Also prints errors as an array to console. (A xml or json tournament object must be already loaded)
 mainTournament.leaderboard(tournamentJsonString?) 
 Takes an optional json string as a parameter.  If given a jsonstring it will first convert it to javascript tournament/player objects. Works off the current javascript Tournament object: sorts first by score then by holes. Returns JSON string.
                                
 Example: leaderboard('{ "tournament": { "name": "British Open", "start date": " ", "end date": " ", "award": 840000, "yardage": 6905, "par": 71, "round": 1, "players": [ {"lastname": "Montgomerie", "firstinitial": "C", "score": -3, "hole": 17 }, {"lastname": "Fulke", "firstinitial": "P", "score": -5, "hole": "finished"} ] } }');
 mainTournament.projectScoreByIndividual(playerJsonString) 
Takes a json string as a parameter (representing a player).  Works off the current javascript Tournament object: Returns the player in JSON with a projected score (By Individual - indivdual rate of progress) 

                            
 Example: projectScoreByIndividual('{"lastname": "Montgomerie", "firstinitial": "C", "score": -3, "hole": 17}')
 mainTournament.projectScoreByHole(playerJsonString) 
Takes a json string as a parameter (representing a player).  Works off the current javascript Tournament object: Returns the player in JSON with a projected score. (By Hole - collective rate of progress) 

                            
 Example: projectScoreByHole('{"lastname": "Montgomerie", "firstinitial": "C", "score": -3, "hole": 17}') 
 mainTournament.projectedLeaderboard(projectScoreByXXXFunction, inputJsonString) 
Required projecScoreByXXX Function as parameter. Takes an optional json string as a parameter (entire tournament json string).  If given a jsonstring it will first convert it to javascript tournament/player objects. Works off the current javascript Tournament object: Returns a leaderboard (like leaderboard function) except sorted based on the function passed to it (projectedScore). 

                            
 Example: projectedLeaderboard(projectScoreByIndividual, '{ "tournament": { "name": "British Open", "start date": " ", "end date": " ", "award": 840000, "yardage": 6905, "par": 71, "round": 1, "players": [ {"lastname": "Montgomerie", "firstinitial": "C", "score": -3, "hole": 17 }, {"lastname": "Fulke", "firstinitial": "P", "score": -5, "hole": "finished"} ] } }')
 mainTournament.getWinner() 
IF the tournament is finished (round = 4, status = complete/completed) returns last name of winner. FYI: Printing leaderboard will show winnings if tournament is compeleted.
